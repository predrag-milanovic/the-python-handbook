# Math

Now that we know how to store and change the value of variables, let's do some math. Python uses familiar symbols for common mathematical operations:

- Addition: `a + b`
- Subtraction: `x - y`
- Multiplication: `c * d`
- Division: `a / b`

## Order of operations

Parentheses let us control the [order of math operations](https://www.mathsisfun.com/operation-order-pemdas.html):

```python
first = 5
second = 7
third = 9
average_value = (first + second + third) / 3
print(average_value)
```

This prints `7`, which is the average of `5`, `7`, and `9`.

## Negative numbers

Negative numbers in Python work the way you probably expect. Just add a minus sign:

```python
my_negative_num = -420
```
