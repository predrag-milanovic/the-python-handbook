# Multi-Variable Declaration

When you need to create several new variables, you can assign them on the same line:

```python
sword_name, sword_damage, sword_length = "Excalibur", 10, 200
```

This is the same as writing:

```python
sword_name = "Excalibur"
sword_damage = 10
sword_length = 200
```

Any number of variables can be declared on the same line. This is most useful when the variables are related to one another, because that makes the code easier to understand.

Code that is easy to understand is called clean code.
