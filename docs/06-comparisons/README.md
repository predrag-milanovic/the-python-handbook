# Comparisons

This chapter explains how to compare values in Python, make decisions with conditionals, and combine boolean expressions.

## Lessons

* [01 - Comparison Operators](./01-comparison-operators.md) - Use `==`, `!=`, `<`, `>`, `<=`, and `>=` to compare values.
* [02 - Comparison Operator Evaluations](./02-comparison-operator-evaluations.md) - Understand how comparisons evaluate and chain together.
* [03 - If Statements](./03-if-statements.md) - Run code conditionally using `if`.
* [04 - If / Else](./04-if-else.md) - Provide alternate execution paths with `else` and `elif`.
* [05 - Boolean Logic](./05-boolean-logic.md) - Combine conditions with `and`, `or`, and `not` for more complex checks.
