# Python Numbers

In Python, numbers without a decimal point are called `integers` — whole numbers like `3` or `-3`.

Python also has `float` values for numbers with a fractional part (decimals). Note that dividing two integers with `/` produces a `float` result:

```python
2 + 1      # 3
2 - 1      # 1
2 * 2      # 4
3 / 2      # 1.5  (a float)
```

Because `/` always returns a [float](https://docs.python.org/3/tutorial/floatingpoint.html), use `//` for integer (floor) division when you specifically want an integer result.
