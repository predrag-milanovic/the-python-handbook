* [08 - Logical Operators](./08-logical-operators.md) - Combine boolean expressions with `and` / `or` in numeric contexts.
* [09 - Not](./09-not.md) - Use the `not` operator to invert boolean values.
* [10 - Binary Numbers](./10-binary-numbers.md) - Represent numbers in base 2 and read binary literals.
