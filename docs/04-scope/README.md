# Scope

This chapter explains how variable and function names are visible (their "scope") in Python programs. Read these short lessons to learn the difference between local and global names, common errors when accessing out-of-scope variables, and when to use the `global` keyword.

## Lessons

* [01 - Scope](./01-scope.md) - Introduces scope and shows how variables defined inside a function are not available outside it.
* [02 - Global scope](./02-global-scope.md) - Explains module-level (global) names and how they can be accessed from functions; includes notes about reading vs modifying globals.
