# List Operations - Concatenate

Concatenating two lists means joining them together. In Python, you can do that with the `+` operator.

```python
total = [1, 2, 3] + [4, 5, 6]

print(total)
# Prints [1, 2, 3, 4, 5, 6]
```

The result is a new list that contains all of the items from both lists in order.
