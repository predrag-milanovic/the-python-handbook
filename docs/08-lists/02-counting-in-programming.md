# Counting in Programming

In programming, counting is a little different from everyday counting.

Instead of starting at `1`, we usually start at `0`.

## Indexes

Each item in a list has an index, which tells you that item's position in the list.

Example:

```python
names = ["Bob", "Lane", "Alice", "Breanna"]
```

- Index `0`: `"Bob"`
- Index `1`: `"Lane"`
- Index `2`: `"Alice"`
- Index `3`: `"Breanna"`

The first item is at index `0`, the second item is at index `1`, and so on.
