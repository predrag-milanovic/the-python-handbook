# Modulo Operator in Python

## Find the Remainder

The [modulo](https://en.wikipedia.org/wiki/Modulo_operation) operator can be used to find the remainder after a division operation. For example, `7` [modulo](https://en.wikipedia.org/wiki/Modulo_operation) `2` is `1`, because `2` can be multiplied evenly into `7` at most `3` times.

```text
2 * 3 = 6
7 - 6 = 1
```

The modulo operator is the percent sign: `%`. It is important to recognize that modulo is not a percentage. It just uses the same symbol.

```python
remainder = 8 % 3
# remainder = 2
```

An odd number is a number that, when divided by `2`, leaves a remainder that is not `0`.
