* [04 - List Length](./04-list-length.md) - Measure how many items a list contains.

* [10 - Modulo Operator in Python](./10-modulo-operator-in-python.md) - Use `%` for remainders and repeating patterns.
* [11 - Slicing Lists](./11-slicing-lists.md) - Pull out a range of items from a list.
* [12 - List Operations Concatenate](./12-list-operations-concatenate.md) - Combine lists with the `+` operator.
* [13 - List Operations Contains](./13-list-operations-contains.md) - Check whether a list contains a value.
* [14 - List Deletion](./14-list-deletion.md) - Remove one item or a slice of items with `del`.
* [15 - Tuples](./15-tuples.md) - Learn how tuples store fixed-size grouped data.
