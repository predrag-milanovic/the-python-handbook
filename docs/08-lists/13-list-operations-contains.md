# List Operations - Contains

You can check whether a value is in a list with the `in` keyword. To check that a value is not in a list, use `not in`.

```python
fruits = ["apple", "orange", "banana"]

print("banana" in fruits)
# Prints True
```

The `in` keyword returns `True` when the value exists in the list and `False` when it does not.

```python
fruits = ["apple", "orange", "banana"]

print("banana" not in fruits)
# Prints False
```

Use `not in` when you want to check for absence instead of presence.
