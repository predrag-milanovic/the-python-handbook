# Indexing Into Lists

Now that you know how to create lists, the next step is accessing specific items.

You access an item by using its index inside square brackets.

Indexes start at `0` for the first item and increase by `1` for each next item.

```python
best_languages = ["JavaScript", "Go", "Rust", "Python", "C"]
print(best_languages[1])
# prints "Go"
```

In this example, index `1` refers to the second item, which is `"Go"`.
