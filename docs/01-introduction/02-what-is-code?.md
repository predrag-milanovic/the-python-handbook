# What is code?

Code is a set of instructions that a computer follows one step at a time. A program can contain just a few instructions, or it can contain thousands.

You already saw one of the most common instructions in Python: `print()`. We can use it to display text in the console, like this:

```python
print("this is a string of text")
```

We can also use `print()` to display numbers. Unlike plain [text strings](https://en.wikipedia.org/wiki/String_(computer_science)), numbers are not surrounded by quotes:

```python
print(42)
```

**Adding numbers**

[Addition](https://en.wikipedia.org/wiki/Addition) is one of the most common things a program can do. This also prints `42`:

```python
print(40 + 2)
```

First, Python calculates the value inside the parentheses. Then it prints the result.
