- [02 - What is code?](02-what-is-code?.md) — What code means in practice; using `print()` to show text and numbers, and a simple example of addition.
- [03 - Multiple instructions](03-multiple-instructions.md) — How programs run instructions in order, demonstrated with multiple `print()` calls.
- [04 - Syntax errors](04-syntax-errors.md) — Defines syntax, gives examples across languages, and distinguishes syntax errors from logic and performance issues.
- [05 - Lesson failure](05-lesson-failure.md) — Explains the difference between **Run** and **Submit**, and the lesson failure penalties and mechanics.
